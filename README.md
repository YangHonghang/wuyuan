# TodoList

项目采用前后端分离模式搭建。

* client:前端
* server:后端
* test:测试模块

## Client:

TodoList客户端

> React +Typescritp+antd

```shell
#安装依赖
yarn install
#启动
yarn sart
```

## Server:

TodoList服务器端

> Typescritp+Koa+GraphQL  
>
> 数据库模块是模拟的

```shell
#安装依赖
yarn install
#启动
yarn server
```

## Test:

TodoList测试模块

> Typescript+mocha+chai

```shell
#确保安装过mocha
#安装依赖
yarn install
#启动
yarn test
```
